# INTERNSHIP_TASK_4Overview

An interactive sales dashboard built using Power BI to analyze Sales, Profit, and Growth across time, categories, and regions.

Dataset

File: superstore_sample_dataset.csv
Columns: Order Date, Region, Category, Sub-Category, Sales, Profit, Quantity

Key KPIs
Total Sales
Total Profit
Profit Margin (%)
Sales Growth (%)

Features
KPI cards
Time-series sales & profit analysis
Category and regional performance
Interactive slicers (Date, Region, Category)
Clean and consistent color theme

Tools
Power BI Desktop

Outcome
Provides quick insights into business performance and supports data-driven decisions.
